Projects|01

Flobnb

I built a web app called Flobnb that allows users to book vacation homes and rentals. The app includes functionality to add homes to a cart and clear the cart as needed. Additionally, users can swipe left and right to view different rental properties. The project was built using JavaScript, HTML, and CSS.




Beats by Dre

I built a mock website for Beats, similar to their official headphones page, using React. I incorporated a hamburger menu animation using an npm library and utilized a "products.json" file to loop over the available products. I implemented React state to change the headphones' color when a user clicks on the color option. Additionally, I created an error page that routes the user back to the homepage. Finally, I used React Router to enable navigation between pages.




Calculator App

I built a fully functional calculator app using React. The app enables users to perform basic arithmetic operations such as addition, subtraction, multiplication, and division, as well as more advanced operations like square roots and percentages. I also added a clear button to reset the calculator and a decimal point button for more precise calculations.




Tony's Chocolonely

I created a mock site for Tony's Chocolonely using HTML, SCSS, and JavaScript. The site includes a landing page, an ingredients page that lays out all the ingredients used in the chocolate, and a mission page that explains the company's mission to end slavery in the chocolate industry. Additionally, I implemented a star rating system to allow users to rate the products.
